# Erfindung-Check: Interaktiver Prozess

Ein strukturierter 8-Schritte-Prozess, um zu prüfen, ob deine Idee neu ist und wie du sie von bestehenden Lösungen abgrenzen kannst.

---

## Übersicht der Schritte

| Schritt | Thema | Beschreibung |
|---------|-------|--------------|
| [0. Einleitung](00-Einleitung.md) | Was ist eine Erfindung? | Kontext und Ziel des Checks |
| [1. Kategorie wählen](01-Kategorie-waehlen.md) | Kategorie wählen | Produkt, Material, Software oder Medizin? |
| [2. Markt scannen](02-Markt-scannen.md) | Markt scannen | Ähnliche Produkte und Lösungen finden |
| [3. Patente prüfen](03-Patente-pruefen.md) | Patente prüfen | Was ist bereits geschützt? |
| [4. Publikationen prüfen](04-Publikationen-pruefen.md) | Publikationen prüfen | Was ist bereits veröffentlicht? |
| [5. Treffer bewerten](05-Treffer-bewerten.md) | Treffer bewerten | Was ist wirklich ähnlich? |
| [6. Kern schärfen](06-Kern-schaerfen.md) | Kern schärfen | Kernidee formulieren |
| [7. Ergebnis ableiten](07-Ergebnis-ableiten.md) | Ergebnis ableiten | Einordnung und nächste Schritte |

---

## Ergebnis des Prozesses

Am Ende erhältst du eine erste Einordnung:

- **Neu wirkt plausibel** – keine sehr nahen Treffer
- **Unklar** – nahe Treffer, Abgrenzung möglich oder noch offen
- **Eher bekannt** – Funktion und Mechanismus bereits abgedeckt

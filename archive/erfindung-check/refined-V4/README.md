# Erfindung-Check V4 – Step-by-Step Design

Design-Refinement mit echtem Schritt-für-Schritt-Wizard.

## Änderungen gegenüber V3

- **Ein Schritt auf einmal** – Fokus auf den aktuellen Schritt
- **Single Progress Bar** – keine doppelte Navigation mehr
- **Lucide Icons** – lightbulb, alert-triangle, search, file-search, scale, target, check-circle
- **Responsive** – Mobile-first, touch-freundliche Buttons
- **Design Template** – nutzt `design-templates/step-wizard/`
- **Keine Pfeile in Buttons** – nur „Weiter" und „Zurück"

## Design-Vorlage

Die Step-Wizard-Vorlage liegt unter:
`Process-Optimizer/design-templates/step-wizard/`

- `step-wizard.css` – wiederverwendbare Styles
- `step-wizard.js` – Schritt-Navigation
- `template.html` – Minimalbeispiel für neue Projekte

## Öffnen

`erfindung-check.html` im Browser öffnen. Die Pfade zu CSS/JS sind relativ zum Projektroot.

# Figma Import

One-page versions of all guides for importing into Figma via html.to.design.

## Serve locally

From this folder, run:

```
npx serve .
```

Then open http://localhost:3000 (or the port shown) in your browser.

## Import into Figma

1. Install the [html.to.design](https://www.figma.com/community/plugin/1159123024924461424) plugin
2. Run `npx serve .` in this folder
3. In Figma: Plugins → html.to.design → URL tab
4. Paste: http://localhost:3000/index.html (or /erfindung-check.html, /patent-next-steps.html, /erfindungsbeschreibung.html)
5. Click Import

# Schritt 2: Markt scannen

## Deine Aufgabe

So gehst du vor
Produkte, Demos und Anbieter liefern die **schnellste Einordnung** und die **besten Suchbegriffe**.
**Öffne** maximal 3 Suchorte: Google (Web), YouTube, 1 großer Shop (z.B. Amazon oder Fachshop deiner Branche).
**Nutze** diese Suchmuster (an deine Idee anpassen): `"Schnellspannzwinge Fahrradlenker"`, `"quick release clamp handlebar"`, `"one hand quick release bike clamp"`.
**Notiere** pro Treffer: **anders** / **ähnlich** / **sehr ähnlich** / **praktisch gleich** (ein Satz reicht).
*Am Ende hast du:* Liste mit 5–10 Produkten/Lösungen + Markierung + verwendete Begriffe (Deutsch/Englisch).

Wenn du wenig Zeit hast

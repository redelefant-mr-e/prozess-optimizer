# Patent Next Steps: AI Briefing (No Prior Context Required)

## What This Is

A 7-step process to guide inventors (especially with academic background) through the essential steps of the patenting process. Output: **clear next action** + **decision checklist** tailored to context (Uni, Firma, Startup).

## Core Concepts

- **Erfindungsberater:in:** Invention advisor at universities – supports inventors in coordinating next steps (TTO, patent attorney, etc.)
- **Meldepflicht:** Obligation to report inventions to employer (Uni, Firma)
- **Publication trap:** Publishing before patent application can destroy patent rights
- **TTO:** Technology Transfer Office (at universities)
- **Schutzstrategie:** Patent vs. publication vs. trade secret vs. combination

---

## The 7 Steps

| Step | Do This | Output |
|------|---------|--------|
| **0** | Explain what happens after invention; three scenarios (Uni, Firma, Startup) | User ready |
| **1** | Ask user to identify context: Uni / Firma / Startup. Check Meldepflicht | Context + Meldepflicht yes/no |
| **2** | Identify first contacts: TTO, Patentstelle, Vorgesetzte. Create contact list with priority | Contact list + priority |
| **3** | Explain publication trap; deadlines; checklist before any publication | Checklist: what before publication? |
| **4** | Compare Patent vs. Publication vs. Trade Secret. Help choose rough strategy | Rough strategy |
| **5** | Set priorities: Meldung → Beratung → Strategie → Anmeldung | Prioritized to-do list |
| **6** | Derive next action; when to involve patent attorney | Next action + decision |

---

## Context-Specific Logic

| Context | First contact | Meldepflicht |
|---------|---------------|--------------|
| **Uni/Forschungsinstitut** | Erfindungsberater:in, TTO, Erfindungsmeldestelle – they support inventors in coordinating next steps | Yes (typically) |
| **Firma mit F&E** | Patentstelle, Vorgesetzte | Often yes |
| **Startup/Solo** | IHK, Patentanwalt (Erstgespräch) | No |

## Key Rule

**Never publish before patent application** – in most jurisdictions, public disclosure destroys patent rights.

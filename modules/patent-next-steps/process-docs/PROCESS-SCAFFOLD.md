## Mapping pro Schritt

### Schritt 0: Einleitung

**Zweck:** Überblick geben. Was passiert nach der Erfindung? Warum dieser Guide?

| Element | Inhalt |
|---------|--------|
| Was dieser Guide macht | Kurze Erklärung; drei Szenarien (Uni, Firma, Startup) |
| Wichtige Grenze | Keine Rechtsberatung; erste Orientierung |
| Nächster Schritt | Link zu Schritt 1 |
| Besonderheiten | Kein Ziel, kein "So gehst du vor", kein Tipp. Rein konzeptuell. |

---

### Schritt 1: Standort klären

**Zweck:** Kontext ermitteln. Wo bist du? Wer besitzt die Erfindung? Meldepflicht?

| Element | Inhalt |
|---------|--------|
| So gehst du vor | 2–3 nummerierte Schritte |
| Tabelle | Uni / Firma / Startup → Meldepflicht, typischer Ansprechpartner |
| Fazit | Output: Kontext + Meldepflicht ja/nein |
| Nächster Schritt | Link zu Schritt 2 |

---

### Schritt 2: Erste Ansprechpartner:innen

**Zweck:** Wen kontaktieren? TTO, Patentstelle, Vorgesetzte.

| Element | Inhalt |
|---------|--------|
| Ziel | Einzeiler: Kontaktliste + Priorität |
| So gehst du vor | Wann wen; Kontakt-Checkliste |
| Tipp | Typische Stolpersteine (z.B. falsche Reihenfolge) |
| Fazit | Output: Kontaktliste + Priorität |
| Nächster Schritt | Link zu Schritt 3 |

---

### Schritt 3: Zeitfenster beachten

**Zweck:** Deadlines, Publication trap.

| Element | Inhalt |
|---------|--------|
| Ziel | Einzeiler: Checkliste vor Veröffentlichung |
| So gehst du vor | Publication trap; was vor Patentanmeldung; typische Fristen |
| Tipp | Publication trap – wenn-dann |
| Fazit | Output: Checkliste |
| Nächster Schritt | Link zu Schritt 4 |

---

### Schritt 4: Schutzstrategie wählen

**Zweck:** Patent vs. Publikation vs. Trade Secret – grobe Strategie.

| Element | Inhalt |
|---------|--------|
| Ziel | Einzeiler: Grobe Strategie |
| So gehst du vor | Kurzvergleich; wann welche Strategie |
| Tipp | Typische Fehlentscheidungen |
| Fazit | Output: Grobe Strategie |
| Nächster Schritt | Link zu Schritt 5 |

---

### Schritt 5: Prioritäten setzen

**Zweck:** Was zuerst? Reihenfolge der Schritte.

| Element | Inhalt |
|---------|--------|
| Ziel | Einzeiler: Priorisierte To-do-Liste |
| So gehst du vor | Reihenfolge: Meldung → Beratung → Strategie → Anmeldung |
| Tipp | Typische Reihenfolge-Fehler |
| Fazit | Output: Priorisierte To-do-Liste |
| Nächster Schritt | Link zu Schritt 6 |

---

### Schritt 6: Ergebnis

**Zweck:** Konkrete nächste Aktion. Wann Patentanwalt? Ausstiegsoptionen.

| Element | Inhalt |
|---------|--------|
| Ziel | Einzeiler: Next Action + Entscheider einbeziehen |
| So gehst du vor | Vorarbeit nutzen → Next Action ableiten → wann Patentanwalt |
| Fazit | Output-Checkliste |
| So geht's weiter | Tabelle (Option | Wann?) – Ausstiegsoptionen |
| Besonderheiten | Kein Tipp. "So geht's weiter" statt "Nächster Schritt". Prozess-Ende. |
